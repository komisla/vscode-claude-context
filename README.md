# vscode-claude-context

> VSCode extension — live Claude Code context window monitor

Shows you what is consuming tokens in your current Claude Code session, directly in VS Code:

- **Status bar:** traffic-light indicator (green / yellow / red) based on context fill level
- **Breakdown panel:** token usage by category (system prompt, tools, memory, conversation)
